CSS Table

Cascading Style Sheets or also known as CSS, is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

font-size: Specifies the font size of text CSS

color: Sets the color of text CSS
border: A shorthand property for border-width, border-style and border-color CSS
